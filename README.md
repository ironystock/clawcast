# Streaming OBS Bootstrap (OpenClaw Skill)

Reusable OpenClaw skill for bootstrapping and validating OBS streaming scene packs via `agentic-obs` + `mcporter`.

## What it includes

- Remote/local OBS host targeting
- Workspace-hosted overlay HTTP serving (LAN-safe)
- Baseline scene rebuild automation
- Walkthrough recording smoke test
- Transition presets
- Optional audio baseline helper
- Optional stream dry-run helper
- Troubleshooting and networking references

## Install as skill

Use the folder contents as a skill, or package via your preferred workflow.
